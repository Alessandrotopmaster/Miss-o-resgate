# 🎮 Missão Resgate / Rescue Mission

**Jogo de sobrevivência, estratégia e resgate para até 50 jogadores**
Modos: Solo, Dupla ou Esquadrão | Duração: 12 a 15 min por rodada
Leve, compatível com celulares Android e PC

---

## 🎯 SOBRE O JOGO
Lutar é necessário, mas **resgatar é a verdadeira missão**.
Derrote ameaças, conquiste equipamentos, salve feridos, liberte prisioneiros e busque almas perdidas até no Abismo. O ciclo nunca acaba — sempre há uma nova chance de salvar.

## ⚙️ PRINCIPAIS SISTEMAS
✅ Dois tipos de inimigos (reais e dominados)
✅ Estados do jogador: Saudável → Ferido → Preso → Abismo
✅ Armas só em bases inimigas
✅ Fenômenos naturais e surpresas constantes
✅ Reviravolta final: Portal do Abismo
✅ Ciclo infinito de rodadas

## 📂 ESTRUTURA DO PROJETO

- `Assets/`: Recursos visuais e sonoros
- `Scripts/`: Código fonte
